# LearnTS

A simple backend project to learn and practice TypeScript with Node.js.

---

## Tech Stack

- TypeScript
- Node.js
- Express

---

## Setup

1. Clone the repo
```

git clone [https://github.com/rohitbytecode/learnTS.git](https://github.com/rohitbytecode/learnTS.git)
cd learnTS

```

2. Install dependencies
```

npm install

```

3. Run the project
```

npm run dev

```

---

```

src/
├── modules/

```
